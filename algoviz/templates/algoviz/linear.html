  • TIME COMPLEXITY = O(n)

  • Linear search is a straightforward algorithm for locating a specific element in a list or array.

  • The list's elements are iterated through by the linear search algorithm, which compares each one to the desired value.

  • The algorithm returns the element's index in the list if a match is discovered.

  • The algorithm iterates until the end of the list is reached, signaling that the element is not in the list, if the target element is not present in the list.

  • Implementing linear search is simple and it can be used on both sorted and unsorted lists.

  • On the other hand, its time complexity is O(n), where n is the list's element count.

  • This means that the length of time it takes to find an element in the list grows linearly with its size.

  • Although effective for small datasets, binary search and other more sophisticated search algorithms may yield faster results when applied to larger datasets.
