  • TIME COMPLEXITY = O(v + e)

  • Depth-first search (DFS) is a fundamental graph traversal algorithm that explores each branch as far as possible before backtracking.

  • Before turning around, it begins at a starting node and proceeds as far as it can down each branch.

  • The algorithm tracks the visited nodes and the path taken using a stack or recursion.

  • A node is considered explored when it is visited, and the algorithm then visits its unexplored neighbors recursively.

  • The algorithm returns to the closest node with unexplored neighbors and resumes the exploration if it reaches a dead end.

  • DFS can be implemented using recursion or an explicit stack data structure.

  • Unlike breadth-first search, DFS does not guarantee finding the shortest path.

  • It is often used to detect cycles in graphs, topological sorting, solving puzzles like mazes, and in various applications of graph theory.

  • DFS can also be employed in backtracking algorithms, where it systematically explores all possible solutions.

  • Its simplicity and versatility make it a powerful tool for solving graph-related problems in computer science and other fields.
